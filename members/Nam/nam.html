A simple Memory Game

The Memory Game is an interactive web-based game developed using HTML5, CSS3, JavaScript, and React JS. The objective of the game is to match pairs of cards by remembering their positions. It challenges the player’s memory and focus. The game dynamically renders a shuffled grid of cards and tracks the player's moves and matched pairs in real-time. I used React state hooks like useState and useEffect to manage the game logic, flipping animations, and timing. The layout is fully responsive using CSS Flexbox/Grid, making it playable on both desktop and mobile devices. This project strengthened my skills in React component structure, conditional rendering, and handling user events efficiently.
